# ACC102_Individual_task
## Firm_Performance_Analysis

## 1. Introduction

This project examines the relationship between firm size, financial leverage, and profitability using financial data from Chinese listed companies. The goal is to understand how firm characteristics influence financial performance and support better investment decisions.

---

## 2. Data Source

The data is obtained from the CSMAR database and includes key financial indicators such as:

* Net Profit
* Return on Assets (ROA)
* Return on Equity (ROE)
* Leverage
* Profit Margin

Firms are grouped into different size categories: Low, Medium-Low, Medium-High, and High.

---

## 3. Methodology

### Data Visualisation

Bar charts are used to compare financial performance across firm size groups.

### Regression Analysis

An OLS regression model is applied to examine the relationship between profitability (ROA), leverage, and profit margin.

---

## 4. Key Findings

* Larger firms generate significantly higher net profits, indicating economies of scale.
* ROA and ROE increase with firm size, showing improved efficiency and returns.
* Leverage rises with firm size, suggesting higher financial risk.
* Smaller firms show negative profit margins in some cases, indicating instability.
* Regression results suggest a positive relationship between leverage and ROA, but results are not statistically significant due to the small sample size.

---

## 6. Limitations

* The dataset contains only four observations (firm size groups).
* Regression results lack statistical significance.
* Findings should be interpreted with caution.

---

## 7. Conclusion

The analysis shows that larger firms achieve higher profitability and efficiency but also face higher financial risk. Investors should consider the trade-off between return and risk when making decisions.
